# Curso R para Políticas Públicas

Prof. Ronaldo Baltar

Em breve, um curso de R para análise de dados em políticas públicas. O curso abordará desde os conceitos básicos de R, passando por manipulação de dados, visualização, até técnicas avançadas de análise estatística e modelagem preditiva. O objetivo é capacitar os participantes a utilizar o R como uma ferramenta para a análise e interpretação de dados em políticas públicas. 

será um curso autoinstrucional, baseado no príncípio (e na minha experiência de décadas) de que a melhor forma de aprender é praticando. O curso terá conteúdos explicativos no formato de ebook e exercícios para serem feitos no próprio R, utilizando o pacote Swirl (https://swirlstats.com/).
